## 六、新手最容易踩的几个坑

**坑1：挂着代理注册**
系统检测到你的IP和填写的国家不匹配，直接砍单，钱自动退回。注册前记得关掉一切代理工具。

**坑2：邮箱填错了**
VPS 的初始密码、账单通知都发到这个邮箱。填错了找不回来，只能联系客服，很麻烦。

**坑3：国家选错了**
必须选 China，这是最硬的限制，选成美国或者其他国家，大概率被判欺诈。

**坑4：不知道官网在哪**
搬瓦工主域名在国内可能被 DNS 污染，访问不了。遇到打不开的情况，试试官方镜像站 bwh81.net，效果一样。

**坑5：忘记用优惠码**
优惠码是循环折扣，不只省一次，以后每次续费也按折扣价，等于永久优惠。填在购物车的 Promotional Code 框里，点 Validate Code 验证一下。
